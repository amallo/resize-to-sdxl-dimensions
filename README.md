
The "Resize Image SDXL" node is an extension to the conventional "Image Resize" node, that resize image to
the nearest allowed dimensions compatible with Stable Diffusion XL.

* 1024 x 1024

* 1152 x 896

* 896 x 1152

* 1216 x 832

* 832 x 1216

* 1344 x 768

* 768 x 1344

* 1536 x 640

* 640 x 1536
